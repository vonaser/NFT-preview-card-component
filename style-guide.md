# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

- Soft blue: hsl(215, 51%, 70%) 	#8bacda
- Cyan: hsl(178, 100%, 50%) 	#00fff7
